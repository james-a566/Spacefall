# Starfall

Starfall is a homebrew NES arcade shooter built in 6502 assembly using ca65/ld65.

## Current Release

**Release build:** v1.0  
**Status:** Complete playable loop

The game includes:
- title screen
- intro/tutorial flow
- 12 level progression
- boss encounters
- final core sequence
- ending screen
- return to title
